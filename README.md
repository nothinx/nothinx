### 🦾 Hexapod Robot Firmware
**[legacy_2025](https://github.com/nothinx/legacy_2025)** &nbsp;·&nbsp; **[program_hexapod_robot_legacy](https://github.com/nothinx/program_hexapod_robot_legacy)**

A modular, high-performance hexapod firmware for **Teensy 4.1** built on a clean 4-layer architecture:

| Layer | Role |
|---|---|
| `HexaServos` | 18-channel PWM via 2× PCA9685 @ 50Hz |
| `HexaKinematics` | 4×4 matrix IK with CMSIS-DSP SIMD |
| `HexaGait` | Smooth Tripod Gait trajectory generator |
| `Hexapod` | Facade API — `robot.walk()`, `robot.goHome()` |
