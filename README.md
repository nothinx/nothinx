### 🦾 Hexapod Robot Firmware
<p><strong><a href="https://github.com/nothinx/legacy_2025">legacy_2025</a></strong> · <strong><a href="https://github.com/nothinx/program_hexapod_robot_legacy">program_hexapod_robot_legacy</a></strong></p>

A modular, high-performance hexapod firmware for **Teensy 4.1** built on a clean 4-layer architecture.

| Layer | Role |
|:---:|:---|
| `HAL` | Hardware abstraction for servos & sensors |
| `Motion` | Inverse kinematics & gait generation |
| `Control` | High-level behavior & state machine |
| `Comm` | Serial/BLE/ROS2 communication bridge |
